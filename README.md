## 🍔Food Delivery App-Inspired UI

This project is a modern food delivery mobile app UI designed using Figma, inspired by the user experience and interface of Swiggy.

## Project Overview

The goal of this project is to design a clean, intuitive, and visually appealing food ordering experience. 
This UI concept focuses on simplicity, usability, and modern design patterns used in real-world applications.

## ✨ Features

- 🔐 Login & Sign Up screens
- 🔗 Social login options
- 🍕 Food categories section
- 🛒 Interactive food cards with pricing
- 🎯 Clean and minimal UI layout
- 🧾 Cart access with floating button

## 🎨 Tools Used

Figma (UI/UX Design)

## 📚 What I Learned

- Structuring mobile app UI layouts
- Improving user experience (UX)
- Creating reusable components in Figma
- Understanding modern UI design principles
